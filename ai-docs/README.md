# ai-docs

Working documents for the builder agent pipeline.

- `prd/`           — the source PRD for this site (drop it here)
- `conventions/`   — CONVENTIONS.md the agent establishes for this project
- `components/`     — per-section build stories
- `verification/`   — build/a11y/responsive verification reports
- `client/`         — client-facing deliverables (e.g. the confirmation checklist .docx)

Feature dependencies the agent proposes from the PRD will be listed in
`DEPENDENCIES.md` at the project root for you to install on approval.
